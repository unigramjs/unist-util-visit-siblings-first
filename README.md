# unist-util-visit-siblings-first

A typescript library

## Installation

```bash
npm install unist-util-visit-siblings-first
```

## Usage

```typescript
import { greet } from 'unist-util-visit-siblings-first';

console.log(greet('World')); // Hello, World!
```
